ブックマークレットの使い方

このページでは、CHUNITHM-NETのベスト枠 / 新曲枠を画像化するブックマークレットの使い方を説明します。

1. コードをコピー

まず、ブックマークレットのコードをコピーしてください。

javascript:(function(){
    const script = document.createElement('script');
    script.src = "https://taka-4602.github.io/chunithm-best-generator/main.js?" + new Date().getTime();
    document.body.appendChild(script);
})();

2. 登録方法を選択

💻 PC (Chromium based / Firefox)

※ Windows, macOS, Linux Distro 共通の手順です。

  1. ブラウザのブックマークバーを右クリックし「ページを追加」や、「ブックマークを追加」を選択します。
  2. 「名前」の欄に分かりやすい名前（例: CHUNITHMツール）を入力することができます。
  3. 「URL」の欄に、先ほどコピーしたコードを貼り付けます。
  4. 「保存」をクリックすれば、ブックマークバーに登録されます。
  5. CHUNITHM-NETを開いた状態で、登録したブックマークをクリックすると実行されます。

🤖 Android (Chrome)

  1. まず、このページ（どのページでも可）を普通にブックマークに追加します。
  2. ブックマーク一覧を開き、追加したブックマークの横にある︙（メニューボタン）をタップして「編集」を選択します。
  3. 「名前」を分かりやすいもの（例: CHUNITHMツール）に変更することができます。
  4. 「URL」の欄の中身を全て消去し、コピーしたコードを貼り付けます。
  5. 右上の←で戻り、保存します。
  6. CHUNITHM-NETを開いた状態で、アドレスバーに今つけた名前を入力し、候補に表示されるブックマーク（★マーク付き）をタップすると実行できます。

📱 iOS (Safari)

  1. まず、このページ（どのページでも可）を普通にブックマークに追加します。
  2. ブックマークを開き、右下の「編集」をタップします。
  3. 先ほど追加したブックマークをタップして編集画面を開きます。
  4. 名前を分かりやすいもの（例: CHUNITHMツール）に変更することができます。
  5. URLが表示されている部分をタップし、中身を全て消去してから、コピーしたコードを貼り付けます。
  6. 「完了」をタップして保存します。
  7. CHUNITHM-NETを開いた状態で、アドレスバーに今つけた名前を入力し、候補に表示されるブックマークをタップすると実行できます。
